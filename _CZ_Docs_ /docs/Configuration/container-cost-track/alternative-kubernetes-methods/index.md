For the most accurate results, CloudZero recommends that you use the [CloudZero Kubernetes Agent](/docs/installation-of-cloudzero-agent-for-kubernetes).

For some use cases that do not require granular Kubernetes data, CloudZero offers the following alternatives to the CloudZero Agent.

* [EKS SCAD Support](/docs/eks-scad-support)
* [GKE Cost Allocations](/docs/gke-cost-allocation)
