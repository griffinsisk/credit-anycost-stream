<Callout icon="ℹ️" theme="info">
  **Source for Recommendations**: Spend data in the CloudZero platform.
  
  **Frequency**: Checked once per day. If the Recommendation is marked as **Ignored**, it will still be updated, but notifications will no longer be sent for any updates.
</Callout>

The following Recommendations target savings on the Azure platform.

* [Purchase Azure PostgreSQL Database Reservations](/docs/postgresql-database-reservations): Created when there is pay-as-you-go spend for Azure Database for PostgreSQL Single Server or Azure Database for PostgreSQL Flexible Server that could be covered by a reservation for compute resources.
* [Purchase Azure Redis Cache Reservations](/docs/redis-cache-reservations): Created when there is pay-as-you-go spend for Azure Cache for Redis that could be covered by a reservation for compute resources.
* [Purchase Azure Premium Disk Reservations](/docs/premium-disk-storage-reservations): Created when there is pay-as-you-go spend for Azure Premium Disk Storage that could be covered by a reservation for storage resources.
* [Purchase Azure MySQL Database Reservations](/docs/mysql-db-reservations): Created when there is pay-as-you-go spend for Azure SQL Database or SQL Managed Instance, or both, that could be covered by a reservation for compute resources.
* [Purchase Azure SQL Database Reservations](/docs/sql-database-reservations): Created when there is pay-as-you-go spend for Azure Database for MySQL Single Server or Azure Database for MySQL Flexible Server, or both, that could be covered by a reservation for compute resources.
