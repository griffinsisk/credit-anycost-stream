<Callout icon="ℹ️" theme="info">
  **Source for Recommendations**: Spend data in the CloudZero platform.
  
  **Frequency**: Checked once per day. If the Recommendation is marked as **Ignored**, it will still be updated, but notifications will no longer be sent for any updates.
</Callout>

The following Recommendations target savings on the GCP platform.

* [Standard Storage Class exceeds target threshold](/docs/excessive-gcp-standard-storage-usage): Focuses on optimizing GCP Cloud Storage classes based on spend.
* [Disk Snapshots exceeds target threshold](/docs/disk-snapshots-exceeds-target-threshold): Created when the cost of GCP Cloud Engine disk snapshots exceeds 5% of the total Compute Engine disk storage cost.
* [Standard Disk below target threshold](/docs/standard-disk-below-target-threshold): Focuses on usage of Standard Persistent Disk storage.
* [Multi-region Storage exceeds target threshold](/docs/excessive-gcp-multi-region-storage-usage): Focuses on optimizing GCP Cloud Storage with Multi-Region bucket location by spend.
* [Supported GCP Recommenders](/docs/gcp-recommender-insights/): CloudZero supports ingestion of GCP Recommender data.
